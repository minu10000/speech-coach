## 🚀 Gemini API 키 발급 방법

### 1 단계: Google AI Studio 접속
- [https://aistudio.google.com/app/apikey](https://aistudio.google.com/app/apikey) 에 접속합니다.
- Google 계정으로 로그인합니다.

### 2 단계: API 키 생성
- **"Create API Key"** 버튼을 클릭합니다.
- 새 프로젝트를 생성하거나 기존 프로젝트를 선택합니다.
- API 키가 생성되면 복사합니다 (형식: `AIzaSy...`)

### 3 단계: API 키 저장
- SpeechCoach 앱에서 우측 상단의 API 상태 배지를 클릭합니다.
- "Gemini API 키" 입력란에 복사한 키를 붙여넣습니다.
- **"저장"** 버튼을 클릭합니다.

✅ **완료!** 이제 Gemini AI 를 사용한 피드백을 받을 수 있습니다.

## 🤖 OpenAI API 키 발급 방법

### 1 단계: OpenAI 플랫폼 접속
- [https://platform.openai.com/api-keys](https://platform.openai.com/api-keys) 에 접속합니다.
- OpenAI 계정으로 로그인합니다.

### 2 단계: API 키 생성
- **"Create new secret key"** 버튼을 클릭합니다.
- 키 이름 (예: `SpeechCoach`) 을 입력합니다 (선택사항).
- **"Create secret key"** 를 클릭합니다.

### 3 단계: API 키 복사
- 생성된 API 키를 복사합니다 (형식: `sk-...`).
- ⚠️ **주의**: 이 키는 한 번만 표시됩니다. 반드시 바로 복사하세요!

### 4 단계: API 키 저장
- SpeechCoach 앱에서 우측 상단의 API 상태 배지를 클릭합니다.
- "OpenAI API 키" 입력란에 복사한 키를 붙여넣습니다.
- **"저장"** 버튼을 클릭합니다.

✅ **완료!** 이제 OpenAI GPT 를 사용한 피드백을 받을 수 있습니다.

## ⚙️ API 키 설정 방법

### 앱에서 설정하기

1. **피드백 페이지** (`feedback.html`) 를 엽니다.
2. 우측 상단의 **API 상태 배지**를 클릭합니다.
3. 모달 창에서 다음을 설정합니다:
   - **사용할 AI API**: Gemini 또는 OpenAI 선택
   - **API 키**: 해당 API 키 입력
4. **"저장"** 버튼을 클릭합니다.

### 초기화하기
- API 키를 삭제하려면 설정 모달에서 **"초기화"** 버튼을 클릭합니다.

## 💰 사용 요금

### Gemini API
- **무료 플랜**: 분당 60 회 요청 (대부분의 개인 사용자에게 충분)
- **유료 플랜**: 더 높은 한도가 필요한 경우
- 자세한 내용: [Gemini API 가격](https://ai.google.dev/pricing)

### OpenAI API
- **종량제**: 사용량에 따라 과금
- `gpt-4o-mini` 모델은 매우 저렴함 (100 만 토큰당 $0.15)
- 일반적인 스피치 피드백 1 회당 약 500-1000 토큰 사용
- 자세한 내용: [OpenAI 가격](https://openai.com/api/pricing/)

---

## 🔒 보안 안내

- API 키는 **브라우저의 localStorage**에 저장됩니다.
- 서버로 전송되지 않으며, 직접 API 를 호출합니다.
- 타인과 API 키를 공유하지 마세요.
- 키가 유출되었다고 생각되면 즉시 재생성하세요.

## ❓ 자주 묻는 질문

### Q. 어떤 API 를 사용해야 하나요?
- **Gemini**: 무료, 빠른 응답, 한국어 잘 지원
- **OpenAI**: 고품질 피드백, GPT-4o 의 우수한 분석력

### Q. API 키를 입력했는데 "연결 안됨"이 표시돼요.
- API 키가 올바른지 확인하세요.
- 브라우저 콘솔 (F12) 에서 오류 메시지를 확인하세요.
- 네트워크 연결 상태를 확인하세요.

### Q. 요금이 너무 많이 나오는데 어떻게 하나요?
- Gemini 의 무료 플랜을 사용하세요.
- OpenAI 는 사용량 제한을 설정할 수 있습니다.

### Q. API 를 변경하려면 어떻게 하나요?
- 설정 모달에서 "사용할 AI API"를 변경하고 저장하세요.

## 🆘 문제 해결

### "API 키가 설정되지 않았습니다" 오류
1. 우측 상단 API 상태 배지 클릭
2. API 키 입력 후 저장

### "네트워크 오류" 발생
1. 인터넷 연결 확인
2. 브라우저 방화벽 설정 확인
3. VPN 사용 중이면 해제

### "유효하지 않은 API 키" 오류
1. API 키를 다시 복사하여 입력
2. 공백이 포함되지 않았는지 확인
3. API 키가 만료되지 않았는지 확인
